## 六、项目结构
### 1、前台（vue-assistant）   
vue项目，使用VS Code进行编写。采用的前端框架为基于Vue 2.0的Vant2移动端组件库，在后续安装项目依赖时要注意。
### 2、后台（vue-backstage）
vue项目，使用VS Code进行编写。采用的前端框架为element
### 3、后端（AssistantServer）  
SSM项目，导入IDEA配置好Tomcat服务器即可运行。包含项目运行所需接口，通过Maven对项目所需依赖包进行管理。
